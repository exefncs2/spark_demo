# S_ML

This is a Spark Machine Learning project.

## Structure

- `base/`: Contains the base engine and core functionalities.
- `center/`: Handles Spark communication and connection.
- `jobs/`: Contains job scripts.
- `model/`: Contains machine learning models.
- `tests/`: Contains test cases.
- `util/`: Contains utility functions.
- `worker/`: Contains worker scripts.
